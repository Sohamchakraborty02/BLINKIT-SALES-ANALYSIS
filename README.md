# Blinkit Sales Dashboard (Power BI)

This project is a Power BI dashboard built to analyze Blinkit's sales data across different outlets, locations, and product categories. It helps in understanding sales trends and customer preferences in a simple visual way.

# Key Insights

* Supermarket Type 1 has the highest sales
* Tier 3 locations contribute the most
* Fruits, Snacks, and Household items are top-selling
* Low Fat products perform slightly better than Regular
* Average rating is around 3.9

# Tools Used

* Power BI
* Excel
* Power Query
* DAX

# Features

* Interactive filters (Location, Size, Item Type)
* KPI cards (Total Sales, Avg Sales, Items, Rating)
* Charts: Bar, Donut, Line
* Easy navigation with reset filters
